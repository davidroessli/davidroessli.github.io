Earlier this year I mentioned that Information Architects Japan (iA) had published their annual Web Trend Map (v4) onto the Tokyo Metro map.

Well, currently iA is building an interactive version of the Web Trend Map under webtrendmap.com which looks cool. A preview is available on YouTube.

This web app is scheduled to launch July 14th. Stay tuned.
