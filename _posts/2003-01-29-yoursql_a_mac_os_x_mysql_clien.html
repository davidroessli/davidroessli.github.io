--- layout: post title: YourSQL, a Mac OS X MySQL client date: '2003-01-29 09:59:55 +0100' excerpt: YourSQL, a new MySQL client for Mac OS X on the block. mt_id: 378 blog_id: 11 post_id: 378 basename: yoursql_a_mac_os_x_mysql_clien categories: - daily --- If you're looking for a simple client to manage your online MySQL database, give YourSQL a try. Personally, I'm still a great fan of myPHPAdmin, but this MySQL client looks quite promising (and it's GPL).

(click to zoom)
It is certainly handy to create a database, edit a table's structure, etc. It's a little less practical when you wish to edit a table's data. It's done inline, a bit like a spreadsheet. I would have preferred a separate window, with one field per line e.g. But let's give it a try.. Need more info on how to install MySQL on your mac ? Go to Marc Liyanage's excellent www.entropy.ch MySQL section for tips, tricks, HOW-TOs and installable software. We've previously mentionned Marc's pages on Macinblog.
